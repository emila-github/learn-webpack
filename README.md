# WebPack的安装 #
## 安装命令 ##
    $ npm install webpack -g
## 使用webpack ##
	`
	$ npm init  # 会自动生成一个package.json文件
	$ npm install webpack --save-dev #将webpack增加package.json文件中`


## 可以使用不同的版本 ##
    $ npm install webpack@1.2.x --save-dev
## 如果想要安装开发工具 ##
    $ npm install webpack-dev-server --save-dev
